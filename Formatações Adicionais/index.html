Principais Formatações

Negrito / Destaque

Podemos utilizar <b> como usada agora.

Mas a forma correta seria com <strong> forma semântica, que tem significado!

Itálico / Ênfase

Podemos utilizar <i> para deixar em itálico.

Mas a forma correta seria <em>, forma semântica!

Formatações Adicionais

Texto Marcado

Podemos usar um texto marcado com <mark>

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado

Texto Inserido

Podemos marcar um texto como inserido para dar um ênfase e indicar que ele foi colocado depois.

Tem tambem a tag <u> que não é uma forma semântica

Texto Sobrescrito

Por exemplo x ao quadrado x20- 4

E por também H2O, que seria H2O
